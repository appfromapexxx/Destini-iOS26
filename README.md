# Destini - iOS 26 Futuristic Edition 🚀

[![Platform](https://img.shields.io/badge/Platform-iOS%2018%2B-blue.svg)](https://developer.apple.com/ios/)
[![Swift](https://img.shields.io/badge/Swift-6.0-orange.svg)](https://swift.org)

Destini 是一款沉浸式的互動故事 App。這不僅僅是一個簡單的選擇遊戲，這是一個經過「iOS 26」概念強化的未來實驗。透過極致的視覺美學與現代化的技術架構，重新定義了在行動裝置上的閱讀體驗。

## ✨ 特色功能

### 💎 未來化視覺設計 (iOS 26 Concept)
*   **Glassmorphism 2.0**: 全面的玻璃擬態介面，具備動態模糊與發光邊框效果。
*   **動態流體背景**: 背景色彩會隨著故事的情感與進度微調，提供極致的沉浸感。
*   **SF Symbols 6 整合**: 為每一個互動提供具備物理擬真感的動態圖示。

### 🏎️ 現代化架構
*   **ID-based Navigation**: 徹底捨棄不穩定的索引系統，改用強型別的節點 ID 跳轉，讓故事邏輯堅如磐石。
*   **Centralized Design System**: 所有的視覺規範（色彩、字體、圓角）均由一套中心化的系統管理，易於擴充。

### 🎭 感官互動
*   **高級物理動畫**: 使用 `UIViewPropertyAnimator` 打造的非線性轉場效果。
*   **精緻觸覺回饋**: 整合 `Haptic Feedback`，讓每一次選擇都能感受到重量。

## 🛠️ 技術組件

*   **UIKit**: 基礎介面開發。
*   **DesignSystem**: 未來風格的封裝。
*   **MVC Pattern**: 標準且強化的軟體設計模式。

## 🎬 快速開始

1. 克隆此專案。
2. 使用 Xcode 打開 `Destini-ios26.xcodeproj`。
3. 選擇任何 iOS 18+ 的模擬器或真機進行執行。
4. 開始你的冒險！
